# SMART NYUKI – Streamlit Edition

Live beekeeping dashboard – weight monitoring with harvest button.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://smart-nyuki.streamlit.app)

### Features
- Real-time hive weight
- Beautiful orange honey design
- Harvest button (resets hive)
- Works with your Node-RED backend

### Deploy instantly
Just connect this repo to [Streamlit Community Cloud](https://share.streamlit.io)
